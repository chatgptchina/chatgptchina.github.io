国内 ChatGPT 注册教程 无需手机验证码【2024】

2024-01-24

ChatGPT 是由 OpenAI 开发的大型语言模型，具有庞大的知识库，可用于回答各种问题，还能帮你写材料，撸代码，堪称当代打工人的打工利器，但凡有啥问题，都可以先问问 ChatGPT！

但是，在国内注册 ChatGPT 具有一定的门槛。

  • 第一，必须能科学上网。注意，必须是国外 IP，如美国、加拿大、新加坡等国家，香港、澳门的 IP 不行。
  • 第二，需要国外邮箱。可以使用 GMail 或者 Proton。
  • 第三，之前的注册需要国外手机号来接收验证码，现在不用了。

一、注册邮箱

GMail：https://support.google.com/mail/answer/56256?hl=zh-Hans
Proton：https://proton.me/

1.1 注册 Proton

第一、选择右上角【Create a free account】

第二、选择免费套餐

第三、选择邮箱后缀，设置用户名和密码，后缀推荐选 proton.me

第四、拖动滑块完成真人验证，设置昵称，然后跳过手机验证。



注册完成，即可登录邮箱，等着接收验证码。

二、注册 OpenAI 账号

再次提醒，注册 OpenAI 账号，一定要打开科学上网。

第一、在 OpenAI 首页，点击【Sign up】进入注册页面。https://chat.openai.com

第二、输入邮箱，然后设置密码。

第三、去邮箱通过验证。

第四、设置名称和生日，然后通过真人验证。

到这里，OpenAI 账号就注册成功了。接下来，可以选择 ChatGPT3.5 开始聊天。

三、升级到 ChatGPT-4

GPT-3.5 不够聪明，而且回答很慢，聊起天来不够爽。GPT-4 更加聪明，而且回答很快，聊起天来很流畅。
那么，如何升级到 GPT-4 呢？

OpenAI 用户获取 GPT-4 的使用权限，需要购买 ChatGPT PLUS 会员，会员的收费标准是每个月 20 美刀。
购买 ChatGPT PLUS 会员后，可以在 Chat 界面选择 GPT-4 进行聊天。
这也太贵了，不过为了提高生产效率，只能冲一波了。

冲完后献上购买教程：
ChatGPT 2分钟升级为 GPT-4

API 开发者用户获取 GPT-4 API 的使用权限，也需要升级为付费帐户。然后，需要预充值 5 美元以上，才能开通 GPT-4 的访问权限。
